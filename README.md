# nonebot_api_paddleocr

✨*基于Nonebot的插件，能够将api版本的Paddle OCR接入QQ使用*✨
  
<div align="left">
  
## 功能

- 支持在群聊和私聊中使用
- 支持连续对话
- 支持中英文和字符
## 安装

## 使用方法

 网页api识别：发送 `/apiocr`，仅支持中英文和数字标点的照片，不可切换语言，发送 `/结束` 即可结束当前用户对话，用户互不影响。
